Local Vue App + Local Flask App + Local PostgreSQL

- Create Virtual Environment
    - python3 -m venv .venv

- Activate Virtual Environment
    - source .venv/bin/activate

- Pip Install Requirements
    pip install -r requirements.txt

- Start Flask App
    python app.py
